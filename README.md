Project Overview

The Banking Performance & Insights Dashboard is a comprehensive data analytics project designed to evaluate financial performance, stakeholder value, and profitability across banking operations.

Built using Tableau and SQL, this dashboard transforms complex financial data into interactive visual insights, enabling better strategic decision-making.

Business Problem

Financial institutions need to:

Monitor premium growth and revenue streams

Analyze stakeholder contributions and asset distribution

Track profitability across regions and tenure

Optimize cost structure and operational efficiency

This dashboard provides a centralized solution to analyze and interpret these key business metrics.

Tools & Technologies Used:

Tableau – Data visualization and dashboard design

Excel/CSV – Data preprocessing
